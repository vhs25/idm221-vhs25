"Russ's Book Reviews" Redesign

My redesign of Russ's Book Reviews website was my second User Interface Design project. I chose this site because it was a blank slate. I thought it would be a fun challenge to create a structure and ways for users to interact with eachother and with Russ. I used Adobe XD to work through multiple iterations of UI for the iPhone, iPad and the computer.
